常用IDE快捷键

Visual Studio Code

  • Trigger Suggestion: Ctrl+Space
  • Format document: Ctrl+Shift+I
  • Show Command Palette: Ctrl+Shift+P
  • Quick Open,Go to File… : Ctrl+P
  • New Window/instance: Ctrl+Shift+N
  • Close Window/instance: Ctrl+W
  • Cut line: Ctrl+X
  • Copy line: Ctrl+C
  • Rename Symbol: F2

the Series of JetBrains IDE

  • 代码自动完成提示：Ctrl+Space
  • 显示意图和快速修复：Alt+Enter
  • 参数信息：Ctrl+P
  • 快速查找文件：Ctrl+Q
  • 生成代码：Alt+Insert
  • 格式化带啊：Ctrl+Alt+L
  • 删除光标所在行：Ctrl+Y
  • 复制当前行或选定的块：Ctrl+D
  • 开始新行：Shift+Enter
  • 历史粘贴：Ctrl+Shift+V
  • 运行：Shift+F10
  • 从编辑器运行上下文配置：Ctrl+Shift+F10
  • 在命令行运行：Ctrl+Shift+X
  • 查找/替换：Ctrl+F/R
  • 跳转到指定类：Ctrl+N
  • 跳转到文件：Ctrl+Shift+N
  • 跳转到符号：Ctrl+Alt+Shift+N
  • 跳转到第几行：Ctrl+G
  • 切换编辑器：Alt+Right/Left
  • 弹出最近编辑文件：Ctrl+E
  • 搜索任何地方：Shift+Shift
  • 查找Action：Ctrl+Shift+A
